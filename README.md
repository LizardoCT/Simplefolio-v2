# SIMPLEFOLIO REACT

## Run Locally

Clone the project

```bash
  git clone https://github.com/LizardoCt/Simplefolio-v2
```

Go to the project directory

```bash
  cd Simplefolio-v2
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
